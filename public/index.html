PODAST steganography demo

Workflow

Hiding data (encryption)

  1. upload an image
  2. choose an image from the list of images
  3. specify the first pixel, data bits and pointer bits (private key)
  4. enter text to be encoded into the image
  5. view resulting image on the right
  6. download resulting image

Extracting data (decryption)

  1. upload encrypted image
  2. specify the first pixel, data bits and pointer bits (private key)
  3. view extracted text
